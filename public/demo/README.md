# Demo Videos

This folder contains demo videos and their thumbnails for the GetQuizzy landing page.

## Files Structure:

### Videos:
1. **GetQuizzyDemoVid.mp4** ✅ - Survey experience demo
2. **survey-builder-demo.mp4** ✅ - Survey builder demo
3. **result-builder.mp4** ✅ - Result experience builder demo
4. **dashboard-overview.mp4** ✅ - Dashboard and responses demo

### Thumbnails (in `/thumbnail` folder):
1. **survey-experience.png** ✅ - Thumbnail for GetQuizzyDemoVid.mp4
2. **survey-builder.png** ✅ - Thumbnail for survey-builder-demo.mp4
3. **result-builder.png** ✅ - Thumbnail for result-builder.mp4
4. **dashboard-overview.png** ✅ - Thumbnail for dashboard-overview.mp4

## Video Specifications:
- Format: MP4
- Recommended resolution: 1920x1080 or 1280x720
- Aspect ratio: 16:9
- Duration: 30-60 seconds each
- File size: Keep under 10MB for optimal loading

## Thumbnail Specifications:
- Format: PNG
- Resolution: Should match video resolution (1920x1080 or 1280x720)
- Aspect ratio: 16:9
- Recommended: Use a clear frame from the video that represents the content

## Usage:
These videos and thumbnails are used in the DemoSection component on the landing page. 
- Videos play inline with standard HTML5 controls
- Thumbnails show as poster images before playing
- Custom play button overlay appears on top of thumbnails
- Play button disappears when video is playing
